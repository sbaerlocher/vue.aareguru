Eine schlanke, voll typisierte Vue-3-Komponente, die die Live-Wassertemperatur
der Aare aus der [Aareguru-API](https://aare.guru) anzeigt — für Bern, Thun,
Interlaken, Biel und weitere Städte entlang des Flusses. Gebaut für die Schweizer
Bäder-Community, "Chume mer go schwümme?" beantwortet sich von selbst.

## In 30 Sekunden eingebunden

{/* DRAFT: npm-Name aus package.json ("vue.aareguru"), Version 2.3.1. */}

Installation via npm, pnpm oder yarn:

<Source
  language="bash"
  dark
  code={`pnpm add vue.aareguru
# oder: npm install vue.aareguru
# oder: yarn add vue.aareguru`}
/>

Komponente importieren und verwenden:

<Source
  language="vue"
  dark
  code={`<script setup lang="ts">
import { AareGuru } from "vue.aareguru";
</script>

<template>
  <p>Die Aare in Bern ist gerade <AareGuru city="bern" /> warm.</p>
</template>`}
/>

Das war's. Kein Setup, kein API-Key, keine Konfiguration nötig.

## Feature-Highlights

- **🌊 Live-Daten** — Echtzeit-Wassertemperatur direkt von der Aareguru-API.
- **🏙️ 8 Städte** — Bern, Thun, Brienz, Interlaken, Biel, Hagneck, Olten, Brugg.
- **🌡️ Celsius & Fahrenheit** — Einheit per Prop umschaltbar.
- **🔁 Auto-Refresh & Caching** — konfigurierbarer Cache, optionales Auto-Update.
- **🛡️ Retry mit Backoff** — robuste Fehlerbehandlung mit exponentiellem Backoff.
- **🎨 Voll anpassbar** — Slots für `default`, `loading` und `error`.
- **🔡 TypeScript-first** — vollständige Typdefinitionen, strict mode.
- **♿ Barrierefrei** — ARIA-Attribute und semantisches HTML.
